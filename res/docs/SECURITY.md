# Security Policy

## Supported Versions

Only the latest version of League Profile Editor is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of this tool seriously. If you find a vulnerability, please report it by opening an issue on GitHub. 

We will acknowledge your report and provide a response as soon as possible.
